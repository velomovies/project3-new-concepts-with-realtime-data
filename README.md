## Set up
**1. Clone repository:**
```
git clone https://github.com/velomovies/project3-new-concepts-with-realtime-data.git
```
**2. Install dependencies:**
```json
{ "devDependencies": {
    "babel-core": "^6.26.0",
    "babel-preset-env": "^1.6.1",
    "gulp": "^3.9.1",
    "gulp-babel": "^7.0.1",
    "gulp-clean": "^0.4.0",
    "gulp-concat": "^2.6.1",
    "gulp-connect": "^5.5.0",
    "gulp-imagemin": "^4.1.0",
    "gulp-open": "^3.0.1",
    "gulp-sass": "^4.0.0",
    "gulp-sequence": "^1.0.0",
    "gulp-uglify": "^3.0.0",
    "compression": "^1.7.2",
    "express": "^4.16.3",
    "nunjucks": "^3.1.2",
    "gulp-nodemon": "^2.2.1",
    "dotenv": "^5.0.1"
  },
  "dependencies": {
    "sockjs-client-node": "^1.0.0",
    "stompjs": "^2.3.3"
  }
}  
```

To install all you have to do:
```
npm install
```

**3. Build and start server:**
```
gulp start
```
To run the server without opening a new tab you simply have to do:
```
gulp
```
